# InvitationToMyWedding

#### 介绍
一个婚礼邀请函的微信小程序，欢迎广大备婚人员使用并推广。

#### 使用说明

1. 代码为小程序的前端代码，后台使用小程序的云开发平台，因为免费；
2. 需要自己申请微信小程序appid，并修改app.js里的appid；
3. 欢迎捐赠

#### 安装需求

1. 微信小程序开发工具
2. nodejs

#### 备注

1. 如何下载wx-server-sdk:

  在云函数的文件夹路径下，cmd运行

    npm install --save wx-server-sdk@latest
